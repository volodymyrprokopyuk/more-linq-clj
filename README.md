# more-linq-clj

[Thinking in LINQ](http://www.apress.com/9781430268451) in Clojure

## Installation

```
$ git clone https://github.com/volodymyrprokopyuk/more-linq-clj.git
```

## Usage

```
$ cd more-linq-clj
$ lein repl
more-linq-clj.core=> (xtest)
```
